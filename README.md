# LatteCar
### An Android RC AI Car Controller


## Summary

This app is for controlling rc car aruduino-based. Is a one-way communication to a server with a fix socket (:5050).


## Output

#### Mode : Collect, Train, Manual
#### JoyY : -100 to 100
#### JoyX : -100 to 100
